# factories/select

## Обзор
Возращает фабрику `selectFactory`
Фабрики для работы с компонентом `Select`

#### props 
- `defaultValue`

#### отдаёт
- `$select`(StoreWritable<string>) - какая опция выбрана в селекте 
- `choosedSelect` - евент, для выбора селекта
Прокидываются в компонент `<Select />`
