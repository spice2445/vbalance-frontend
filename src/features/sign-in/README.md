# Компонент SignInByEmail, SignInByTelegram

## Обзор
Компонент, который возвращает форму и отвечает за валидирование полей, авторизацию пользователя по почте.

## Файлы

- `ui/*`: Основная папка где лежит ui формы для авторизации по почте.
- `model`: Здесь лежит сама форма, валидация и логика авторизации по почте.

## Props
Отсутствуют. Все данные берутся по апи.

## Использование
```jsx
import { ByEmail } from 'features/sign-in';
<ByEmail />
```
